# Analysing-dewali-sales-with-the-help-of-python
A data analytics project focused on exploring customer purchasing patterns using Python. Includes data cleaning, manipulation, and exploratory data analysis (EDA) using Pandas, Matplotlib, and Seaborn. The project identifies key customer segments, top-selling products, and insights that can help improve customer experience and business sales.



Project Highlights:

Cleaned and prepared the dataset by handling missing values and formatting data

Performed exploratory data analysis (EDA) using Pandas, Matplotlib, and Seaborn

Identified potential customer groups across:

States

Occupations

Gender

Age groups

Discovered the most selling product categories and high-demand products

Generated insights that can help improve:

Customer experience

Sales performance

Inventory and demand planning
